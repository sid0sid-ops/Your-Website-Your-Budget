### Customize Colors

Edit `css/styles.css` and modify CSS variables in `:root`:

```css
:root {
    --bg-dark: #0a0a0a;        /* Background */
    --text-light: #ffffff;      /* Text */
    --accent: #00ffff;          /* Accent/highlight */
    --accent-light: #0077ff;    /* Light accent */
}
```

### Change Site Title and Meta

Edit the `<head>` section in `index.html`:

```html
<title>Your Website, Your Budget</title>
<meta name="description" content="Your description here">
<meta name="author" content="Your Name">
<meta property="og:title" content="Your Title">
```
